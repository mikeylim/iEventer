# Screenshots & demo assets

Drop captures here for the main `README.md` to embed.

## Expected files

| Filename | What to capture |
|----------|-----------------|
| `home-light.png` | Home page (signed-in) with the Daily Surprise Pick visible, in **light mode**. ~1280×900 |
| `home-dark.png` | Same, in **dark mode** |
| `onboarding.png` | Onboarding flow with several interest pills selected, sticky bottom bar visible |
| `plan-detail.png` | A plan detail page with 3+ events showing image thumbs and the travel-time ribbon |
| `signin.png` | The sign-in screen with the soft blurred background |
| `golden-path.gif` | 30-second screen recording of: sign in → daily pick → search → add to plan → optimize route. Keep ≤ 8 MB |

## Capture tips

- **Browser:** use a clean profile (no extensions / ad blockers) and hide bookmark bar
- **Window size:** 1440×900 is a good resolution for desktop screenshots
- **Light/Dark:** toggle via the sun/moon icon in the top nav before each shot
- **GIF:** record with [Kap](https://getkap.co/) or [Cleanshot](https://cleanshot.com/), export at 12 fps to keep file size down. Convert to GIF only at the end (PNG/WebP for stills).

## Cropping

Use a 16:9 or 16:10 frame so the embeds line up nicely. Keep the top nav visible in every shot.
